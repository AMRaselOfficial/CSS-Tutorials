Description

The :link pseudo-class matches link elements that are unvisited.

This selector has the form:

:link {
  declaration block
}

Examples

This example shows how to set the font color of all unvisited HTML links to red.

a:link {
  color:red;
}

This example sets specific colors for various link states.

a:link {color:#FF0000}
a:visited {color:#00FF00}
a:hover {color:#FF00FF}
a:active {color:#0000FF}
