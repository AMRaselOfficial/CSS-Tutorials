Description

The @media parser directive is used to indicate that one or more rule sets apply to specific media types. The rules that apply to the specicified media type must be contained within the media block.

This directive has the form:

@media media_type and|not|only (media feature) {
  CSS-Code;
}

Media Types

This table lists the different media types for this directive:

Value Description
all All devices

screen Computer screens

tv Television-type devices

Examples

This example set the background color to red for screen media, and black for print media.

@media screen {
  body {
    background-color: red;
  }
}

@media print {
  body {
    background-color: black;
  }
}
