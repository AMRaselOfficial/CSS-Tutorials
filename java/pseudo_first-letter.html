Description

The :first-letter pseudo-element styles the first character of formatted text.

Only the following properties apply to the :first-letter pseudo-element:

  • background properties
  • border properties
  • clear
  • color properties
  • font properties
  • float
  • line-height
  • margin properties
  • padding properties
  • text-decoration
  • text-transform
  • vertical-align (only if 'float' is 'none')

Examples

This example shows the first-letter selector used to style the first character of a paragraph element.

p:first-letter {
   font-size:x-large;
}
<p>The quick brown fox...</p>

This gives the following result:

The quick brown fox...
