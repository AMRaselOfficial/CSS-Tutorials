Property Description

The empty-cells property sets whether or not to display borders and background on empty cells in a table. This property only applies to the separate borders model.

This property has the following syntax:

empty-cells: hide|show|initial|inherit;

Property Values

Value Description
hide No border or background will be shown on empty cells
show (default) Background and borders are shown on empty cells
initial Specifies that the value of the property should be set to the default value
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example hides the borders and background on empty cells in a table.

table {
  border-collapse:separate;
  empty-cells:hide;
}

<table border="1">
  <tr>
    <td>row 1 col 1</td>
    <td>row 1 col 2</td>
  </tr>
  <tr>
    <td>row 2 col 1</td>
    <td></td>
   </tr>
</table>

This produces the following result:

row 1 col 1 row 1 col 2
row 2 col 1
