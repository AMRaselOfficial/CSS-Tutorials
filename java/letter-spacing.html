Property Description

The letter-spacing property sets the space between characters in text.

This property has the following syntax:

letter-spacing: normal|length|initial|inherit;

Property Values

Value Description
normal (default) No extra space between characters
length Defines the space between characters (negative values are allowed)
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the letter spacing for h1 and h2 elements.

h1 {
  letter-spacing:2px;
}

h2 {
  letter-spacing:-2px;
}

This example shows the results of setting the various letter-spacing values:

letter-spacing:normal
letter-spacing:2px
letter-spacing:-2px
