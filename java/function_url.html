Description

The url() function is used to represent a URL (Uniform Resource Locator).

This function has the form:

url(url)

Note: The path in url() can be either absolute or relative and is case sensitive.

Arguments

Name Description
url Specifies the URL to represent. The URL can be quoted with either two single quote (') or double quote (") characters

Examples

This example shows how to set the background image for a web page:

body {
  background: url("http://www.example.com/background.gif")
}

Browser Support

Chrome Firefox IE Safari Opera
