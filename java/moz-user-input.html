Property Description

The -moz-user-input property specifies whether the element can have input focus.

This property has the following syntax:

-moz-user-input:value

Property Values

Value Description
auto (default) Use the element's default behavior for user-input behavior
disabled Element is disabled and will not respond to user input
enabled Element can respond to user input
none The element is considered to be neither enabled nor disabled
inherit Specifies that the value of the property should be inherited from the parent element

Examples

The following example shows the basic use of this property:

<textarea style="-moz-user-input:disabled;
  -webkit-user-input:disabled;
  user-input:disabled;">
This text cannot be changed
</textarea>
