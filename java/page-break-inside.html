Property Description

The page-break-inside property sets the page-breaking behavior inside an element.

This property has the following syntax:

page-break-inside: auto|avoid|initial|inherit;

Property Values

Value Description
auto (default) Insert a page break inside the element if necessary
avoid Avoid inserting a page break inside the element
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the page-breaking behavior to avoid breaking inside table elements.

p {
  page-break-inside: always;
}
