Browser Compatibility

The browser compatibility for each property is shown in the Browser Support table. The table lists the five major browsers and the icon indicates their support of the property. If present, a number in the column indicates the browser version in which the property was first supported.

Note: The compatibility information indicated is for the latest released version of browser at the time this guide was written. Those browser versions are listed in the table below:

Browser Version
Chrome 39.0.2171.95 (December 9, 2014)
Firefox 34.0.5 (December 1, 2014)
Internet Explorer 11.0.12 (September 9, 2014)
Safari 8.0 (October 16, 2014)
Opera 26.0.1656.60 (December 17, 2014)
