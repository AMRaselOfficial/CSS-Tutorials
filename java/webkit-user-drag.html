Property Description

The -webkit-user-drag property specifies whether the element is draggable.

This property has the following syntax:

-webkit-user-drag: auto|none|element;

Property Values

Value Description
auto Specifies that the browser determines whether the element is draggable or not
none (default) Specifies that the element is not draggable
element Specifies that the element itself is draggable instead of its contents
inherit Specifies that the value of the property should be inherited from the parent element

Browser Support

Chrome Firefox IE Safari Opera

Miscellaneous Information

Inherited: No
Defined In: WebKit Specific
Default Value: none
