Property Description

The -moz-column-rule-style property sets the style of the rule between the columns.

This property has the following syntax:

-moz-column-rule-style: none|dashed|dotted|double|groove|hidden|inset|outset|ridge|solid|initial|inherit;

Note: Any values not specified are set to their default properties.

Property Values

Value Description
none (default) Specifies no rule
dashed (default) Specifies a dashed rule
dotted (default) Specifies a dotted rule
double (default) Specifies a double rule
groove (default) Specifies a 3D grooved rule
hidden (default) Specifies a hidden rule
inset (default) Specifies a 3D inset rule
outset (default) Specifies a 3D outset rule
ridge (default) Specifies a 3D ridged rule
solid (default) Specifies a solid rule
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example specifies that a div will have 2 columns with a dotted, blue rule between them:

div
{
  column-rule-style: dotted;
  -webkit-column-rule-style: dotted; /* Safari and Google Chrome */
  -moz-column-rule-style: dotted; /* Firefox */  
  
  column-rule-color: #0000ff;
  -webkit-column-rule-color: #0000ff; /* Safari and Google Chrome */
  -moz-column-rule-color: #0000ff; /* Firefox */  
}

A div containing some sample text would then be formatted like this:

The Lonely Mountain is a towering solitary peak to the west of Largoneth Castle. It is quite tall, and its proud summit often lies shrouded in low-hanging clouds. However, in olden days, mariners had named it Signal Mountain, for its odd glistening white peak reflected the pale moonlight, providing an unmistakable landmark on unclouded nights.
