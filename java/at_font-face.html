Description

The @font-face parser directive is used to define custom fonts.

This directive has the form:

@font-face
{
  font descriptors
}

Examples

This example specifies a font named "myFirstFont" and the URL where it can be found:

@font-face
{
  font-family: myFirstFont;
  src: url('Sansation_Light.ttf'),
       url('Sansation_Light.eot'); /* IE */
}

Now that the font has been define and is ready for use, the following example sets that font for all div elements:

div
{
  font-family: myFirstFont;
}
