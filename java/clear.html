Property Description

The clear property sets which sides of an element are not allowed adjactent floating elements.

This property has the following syntax:

clear: none|left|right|both|initial|inherit;

Property Values

Value Description
none (default) Allows floating elements on both sides
left No floating elements allowed on the left
right No floating elements allowed on the right
both No floating elements allowed on the left or the right
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example shows how to prevent floating elements on the left or the right side of an image.

img {
  float:left;
  clear:both;
}
