Property Description

The border-right property sets all the right border properties in one declaration and is composed of the border-right-width, border-right-style, and border-right-color properties.

This property has the following syntax:

border-right: width style color|initial|inherit;

Note: Any values not specified are set to their default properties.

Property Values

Value Description
width Specifies the width of the right border. See border-right-width for possible values
style Specifies the style of the right border. See border-right-style for possible values
color Specifies the color of the right border. See border-right-color for possible values
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

The following example shows the basic use of this property:

p {
  border-style: solid;
  border-right:thick dotted #0000ff;
}

<p>This is a paragraph.</p>

This produces the following result:

This is a paragraph.
