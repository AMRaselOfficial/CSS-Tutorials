Property Description

The column-rule-color property sets the color of the rule between the columns.

This property has the following syntax:

column-rule-color: color|initial|inherit;

Property Values

Value Description
color Specifies the color of the column rule. The default value is the color of the element
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example specifies that a div will have 2 columns and a blue rule between them:

div
{
  column-count: 2;
  -webkit-column-count: 2; /* Safari and Google Chrome */
  -moz-column-count: 2; /* Firefox */
  
  column-rule-style: outset;
  -webkit-column-rule-style: outset; /* Safari and Google Chrome */
  -moz-column-rule-style: outset; /* Firefox */  
  
  column-rule-color: #0000ff;
  -webkit-column-rule-color: #0000ff; /* Safari and Google Chrome */
  -moz-column-rule-color: #0000ff; /* Firefox */  
}

A div containing some sample text would then be formatted like this:

The Lonely Mountain is a towering solitary peak to the west of Largoneth Castle. It is quite tall, and its proud summit often lies shrouded in low-hanging clouds. However, in olden days, mariners had named it Signal Mountain, for its odd glistening white peak reflected the pale moonlight, providing an unmistakable landmark on unclouded nights.
