Property Description

For absolutely positioned elements, the bottom property sets the bottom edge of an element to a unit above/below the bottom edge of its containing element.

For relatively positioned elements, the bottom property sets the bottom edge of an element to a unit above/below its normal position.

This property has the following syntax:

bottom: auto|length|%|initial|inherit;

Note: This property has no effect when the position is set to static.

Property Values

Value Description
auto (default) Browser calculates the bottom edge position
length Sets the bottom edge position in px, cm, etc. This value can be negative
% Sets the bottom edge position in % of containing element. This value can be negative
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the bottom edge of the image element to 10px above the bottom edge of its containing element.

img {
  position: absolute;
  bottom: 10px;
}
