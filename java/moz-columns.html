Property Description

The -moz-columns property is a shorthand property that can be used to set the -moz-column-width and -moz-column-count properties.

This property has the following syntax:

-moz-columns: auto|column-width column-count|initial|inherit;

Note: Any values not specified are set to their default properties.

Property Values

Value Description
auto (default) Sets the -moz-column-width and -moz-column-count properties to "auto"
column-width Specifies the width of the columns. See -moz-column-width for possible values
column-count Specifies the number of columns. See -moz-column-count for possible values
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example specifies that a div will have 4 columns, each 100px wide:

div
{
  columns:100px 4;
  -webkit-columns:100px 4; /* Safari and Google Chrome */
  -moz-columns:100px 4; /* Firefox */
}

Browser Support

Chrome Firefox IE Safari Opera
