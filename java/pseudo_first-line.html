Description

The :first-line pseudo-element styles the first line of formatted text.

Only the following properties apply to the :first-line pseudo-element:

  • background properties
  • color properties
  • clear
  • font properties
  • letter-spacing
  • line-height
  • text-decoration
  • text-transform
  • vertical-align
  • word-spacing

Examples

This example shows the first-line selector used to style the first line of a paragraph element.

p:first-line {
   font-size:x-large;
}
<p>The quick brown fox...<br/>
Second line</p>

This gives the following result:

The quick brown fox...
Second line
