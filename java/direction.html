Property Description

The direction property sets the text writing direction.

This property has the following syntax:

direction: ltr|rtl|initial|inherit;

Property Values

Value Description
ltr (default) The text direction is left-to-right (ltr)
rtl The text direction is right-to-left (rtl)
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the text direction as right-to-left.

div {
  direction:rtl;
}
