Property Description

The border-style shorthand property sets all of the border styles in one declaration. This property can accept from one to four values.

border-style: <top> <right> <bottom> <left>|initial|inherit;

If all four values are provided, the border styles are as specified.

border-style: <top> <right/left> <bottom>|initial|inherit;

If three values are provided, right and left border styles are set as middle value.

border-style: <top/bottom> <right/left>|initial|inherit;

If two values are provided, top and bottom border style are set as the first value, right and left border style are set as second value.

border-style: <top/right/bottom/left>|initial|inherit;

If one value is provided, all four border style values are set the same.

Property Values

Value Description
none Specifies no border
hidden The same as "none", except in border conflict resolution for table elements
dotted Specifies a dotted border
dashed Specifies a dashed border
solid Specifies a solid border
double Specifies a double border
groove Specifies a 3D grooved border. The effect depends on the border-color value
ridge Specifies a 3D ridged border. The effect depends on the border-color value
inset Specifies a 3D inset border. The effect depends on the border-color value
outset Specifies a 3D outset border. The effect depends on the border-color value
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the style of all four borders.

p {
  border-style: solid;
}

This example shows the results of setting the various border-bottom-style values:

border-bottom-style: none

border-bottom-style: dotted

border-bottom-style: dashed

border-bottom-style: solid

border-bottom-style: double

border-bottom-style: groove

border-bottom-style: ridge

border-bottom-style: inset

border-bottom-style: outset
