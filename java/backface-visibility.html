Property Description

The backface-visibility property specifies whether or not an element should be visible when not facing the screen.

Note: This property is useful when an element is rotated, and you do not want see to the backside.

This property has the following syntax:

backface-visibility: visible|hidden|initial|inherit;

Property Values

Value Description
visible (default) Specifies that the backside will render as visible
hidden Specifies that the backside will render as hidden
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example shows a few different appearance settings:

<style type="text/css"> 

div.test
{
  position:relative;
  height:50px;
  width:50px;
  padding: 4px;
  color: white;
  text-align: center;
  background-color: blue;
  transform:rotateY(180deg);
  /* Firefox */
  -moz-transform:rotateY(180deg);
  /* Chrome and Safari */
  -webkit-transform:rotateY(180deg);
}

#div1
{
  /* IE */
  -ms-backface-visibility:hidden;
  /* Firefox */
  -moz-backface-visibility:hidden;
  /* Chrome and Safari */
  -webkit-backface-visibility:hidden;
}
#div2
{
  /* IE */
  -ms-backface-visibility:visible;
  /* Firefox */
  -moz-backface-visibility:visible;
  /* Chrome and Safari */
  -webkit-backface-visibility:visible;
}
</style>

<div id="div1" class="test">DIV 1</div>
<br />
<div id="div2" class="test">DIV 2</div>

This produces the following result:

DIV 2
