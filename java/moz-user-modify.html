Property Description

The -moz-user-modify property specifies which part of the contents of an element can be modified by the user.

This property has the following syntax:

-moz-user-modify:value

Property Values

Value Description
read-only (default) User can view/select/copy information in the element, but cannot modify the contents
read-write User can view/select/copy and edit the contents of the element
write-only User can select/edit element contents, but cannot view it
inherit Specifies that the value of the property should be inherited from the parent element

Examples

The following example shows the basic use of this property:

<pre style="-moz-user-modify:read-only;
  -webkit-user-modify: read-only;">
This text cannot be changed
</pre>

This produces the following result:

This text cannot be changed

Browser Support

Chrome Firefox IE Safari Opera
