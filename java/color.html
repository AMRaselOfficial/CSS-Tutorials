Property Description

The color property sets the color of text.

This property has the following syntax:

color: color|initial|inherit;

Property Values

Value Description
color Specifies the text color. Refer to the Colors section for the different ways to specify a color
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

These examples set the text-color to blue for different elements.

h1 {
  color: blue;
}

h2 {
  color: #0000ff;
}

p {
  color: rgb(0,0,255);
}
