Property Description

The -ms-interpolation-mode property specifies the interpolation algorithm for scaling an image.

This property has the following syntax:

-ms-interpolation-mode: value;

Note: This property is supported in Internet Explorer from version 7.

Property Values

Value Description
bicubic Specifies the use of a complex interpolation algorithm to make higher quality large images by processing small ones
nearest-neighbor Simple interpolation algorithm to enlarge images. The image quality is lower than with the use of bicubic, but the process is faster

Browser Support

Chrome Firefox IE Safari Opera
