Keyword Description

The initial keyword specifies that a property should be set to its' default value.

This keyboard has the following syntax:

property: initial;

Examples

This example specifies that a div will have 2 columns:

div
{
  color: blue;
}

h3 {
  color: initial;
}

This produces the following result:

This paragraph text is blue

This code text is also blue

This H3 text is black (default color)
