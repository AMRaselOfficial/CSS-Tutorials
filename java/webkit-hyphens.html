Property Description

The -webkit-hyphens property specifies the line breaking rules when line-wrapping.

This property has the following syntax:

-webkit-hyphens: auto|manual|none;

Note: There are two Unicode characters that can be used to manually specify potential line break points within text:

  • U+2010 (HYPHEN) - The "hard" hyphen character indicates a visible line break opportunity (the hypen is rendered even if the line is not broken at that point)

  • U+00AD (SHY) - The "soft" hyphen character indicates the location of a possible line break, but the character is not rendered (in HTML, use &shy; to insert a soft hyphen)

Property Values

Value Description
auto Specifies that line breaks will occur automatically at appropriate hyphenation points, following rules for the selected language

Note: This setting's behavior depends on the language specified using the HTML lang attribute.
manual (default) Specifies that words are broken for only where characters inside the word suggest a line break opportunity
none Specifies that lines will only wrap at whitespace

Browser Support

Chrome Firefox IE Safari Opera
