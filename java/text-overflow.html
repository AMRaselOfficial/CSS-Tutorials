Property Description

The text-overflow property specifies how to handle the overflowed area of text.

This property has the following syntax:

text-overflow: clip|ellipsis|string|initial|inherit;

Property Values

Value Description
clip (default) Specifies that the browser should clip the overflowing text content
ellipsis Specifies that an ellipsis ("...") should be shown for overflowing text content
string Specifies to use the given string to represent the clipped text
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Note: text-overflow: ellipsis comes into play only when:

  • the box has overflow other than visible (with overflow: visible the text simply flows out of the box)
  • the box has white-space: nowrap or a similar method of constraining the way the text is layed out

Examples

The following example shows the basic use of this property:

div.test
{
  white-space:nowrap;
  overflow:hidden;
  width:12em;
  border:1px solid #000000;
}

<p>"text-overflow:ellipsis":</p>
<div class="test"
  style="text-overflow:ellipsis;">
  This is some long text that will not fit in the box
</div>


<p>"text-overflow:clip":</p>
<div class="test"
  style="text-overflow:clip;">
  This is some long text that will not fit in the box
</div>

This produces the following result:

"text-overflow:ellipsis":

Long text that does not fit in the box


"text-overflow:clip":

Long text that does not fit in the box
