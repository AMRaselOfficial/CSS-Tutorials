Property Description

The z-index property sets the stack order of an element.

This property has the following syntax:

z-index: auto|number|initial|inherit;

An element with greater stack order is always in front of an element with a lower stack order.

Note: z-index only works on positioned elements (position:absolute, position:relative, or position:fixed).

Property Values

Value Description
auto (default) Sets the stack order equal to its parents
number Sets the stack order of the element (negative numbers are allowed)
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

The following example shows the basic use of this property:

div#redBox
{
  border:1px solid #333333;
  background-color: red;
  width:150px;
  height:100px;
  position:absolute;
}

div#yellowBox
{
  border:1px solid #333333;
  background-color: yellow;
  width:150px;
  height:100px;
  position:absolute;
  opacity:0.5;
}

<div id="redBox"
  style="top:20px;left:20px;z-index:1;">
  redBox: z-index = 1</div>
<div id="yellowBox"
  style="top:0px;left:0px;z-index:0;">
  yellowBox: z-index = 0</div>
<div id="yellowBox"
  style="top:40px;left:40px;z-index:2;">
  yellowBox: z-index = 2</div>

This produces the following result:

redBox: z-index = 1
yellowBox: z-index = 0
yellowBox: z-index = 2
