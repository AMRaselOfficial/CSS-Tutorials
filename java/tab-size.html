Property Description

The tab-size property specifies the length of the space used for the tab character.

This property has the following syntax:

tab-size: number|initial|inherit;

Property Values

Value Description
number Specifies the number of space-characters that should be displayed for each tab-character (default = 8)
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example sets the tab size of the pre element:

pre {
  tab-size: 4;
  -moz-tab-size: 4;
  -o-tab-size: 4;
}
