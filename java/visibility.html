Property Description

The visibility property sets whether or not an element is visible.

This property has the following syntax:

visibility: visible|hidden|collapse|initial|inherit;

Note: Elements that have visibility set to hidden still take up space on the page.

Property Values

Value Description
visible (default) The element is visible
hidden The element is invisible
collapse Collapse removes a row or column from a table, but it does not affect the table layout
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example shows how to make h1 elements invisible.

h1.visible
{
  visibility: visible
}

h1.hidden
{
  visibility: hidden
}

<h1 class="visible">
  This heading is visible</h1>
<h1 class="hidden">
  This heading is hidden</h1>
<h1 class="visible">
  This heading is visible</h1>

This produces the following result:

This heading is visible

This heading is hidden

This heading is visible
