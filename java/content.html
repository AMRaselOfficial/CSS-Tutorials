Property Description

The content property is used in conjunction with the :before and :after pseudo-elements, to insert generated content.

This property has the following syntax:

content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit;

Property Values

Value Description
none The pseudo-element is not generated
normal (default) The pseudo-element is not generated when used with :before or :after
none Specifies to set the content to nothing
count Specifies to set the content as a counter
attr(attribute) Specifies to set the content as one of the selector's attribute
string Specifies to set the content to the text as specified
open-quote Specifies to set the content to be an opening quote
close-quote Specifies to set the content to be a closing quote
no-open-quote Specifies to remove the opening quote from the content, if specified
close-quote Specifies to remove the closing quote from the content, if specified
url Specifies to set the content to some kind of media (image, sound, video, etc.)
initial Specifies that the value of the property should be set to the default value
inherit Specifies that the value of the property should be inherited from the parent element

Examples

This example inserts the text "Click Here: " at the beginning and as part of the link.

a:before {
  content:"Click Here: ";
}
