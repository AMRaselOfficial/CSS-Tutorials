Keyword Description

The inherit keyword specifies that a property should inherit the value of the parent element.

This keyword has the following syntax:

property: inherit;

Examples

This example specifies a div that inherits the background-color:

div
{
  padding: 10px; // just for spacing
}

.inherit-div
{
  background-color: inherit;
  border: solid 1px black;
}

<div class="inherit-div">
  <small>DIV by itself - background is gray</small>
</div>
<br/>
<div style="background-color: lightblue;">

<small>This DIV's background is "lightblue"</small>
  <div class="inherit-div">
    <small>DIV as child - background is lightblue</small>
  </div>
</div>

This produces the following result:

DIV by itself - background is gray

This DIV's background is "lightblue"
DIV as child - background is lightblue
