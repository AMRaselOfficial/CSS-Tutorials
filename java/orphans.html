Property Description

The orphans property sets the minimum number of lines to be left at the bottom of a page when a page break occurs inside an element.

Examples

This example ensures that at least three lines of text will be displayed at the bottom of the page when a page break occurs inside a paragraph.

p {
  orphans:3;
}
